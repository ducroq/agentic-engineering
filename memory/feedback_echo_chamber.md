Case studies where a framework is applied to verify itself risk being an echo chamber (e.g., agent-ready-papers verification framework applied to a report about verification).

**Why:** The user flagged issue #1 (Digital Engineers case study) as potentially circular — the verification framework verifying itself doesn't constitute independent evidence.

**How to apply:** When proposing new case studies or evaluating evidence, check whether the evidence source is independent of the pattern being demonstrated. Flag circular evidence explicitly. Prefer case studies where the tool/pattern is applied to a genuinely external domain.
